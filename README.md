🚀 Jornada DEV START 2026 — TOTVS Paulista

Este repositório foi criado para centralizar e organizar todos os exercícios, desafios e projetos práticos desenvolvidos durante a Jornada DEV, a formação oficial do Programa START 2026 da TOTVS Paulista.

O objetivo desta jornada é construir uma base sólida em programação, partindo do zero absoluto até o desenvolvimento de aplicações reais voltadas para o ecossistema do Protheus, o maior ERP do Brasil.
📅 Estrutura do Programa & Conteúdo

O aprendizado está dividido em duas grandes etapas, refletidas na organização deste repositório:
🧩 Parte 1: Fundamentos da Programação

Foco em lógica de programação e estruturação de pensamento algorítmico utilizando a linguagem Harbour.

    Lógica de programação e algoritmos

    Variáveis e tipos de dados

    Estruturas de decisão e repetição

    Funções e modularização

    Arrays e manipulação de coleções de dados

🏢 Parte 2: Protheus & ADVPL

Imersão no ambiente real de desenvolvimento da TOTVS, aplicando a teoria na prática do mercado corporativo.

    Arquitetura e tabelas do Protheus

    Telas de cadastro (CRUD completo)

    Criação de menus e rotinas customizadas

    Desenvolvimento de relatórios operacionais

🛠️ Tecnologias Utilizadas

    Harbour / ADVPL: Linguagens foco da formação.

    Protheus ERP: Ambiente de aplicação prática.

    Git & GitHub: Para controle de versão e portfólio.

📝 Sobre o Repositório

Cada pasta dentro deste repositório representa um módulo, dia de aula ou lista de exercícios específicos. O histórico de commits reflete a minha evolução contínua ao longo de toda a formação até o desenvolvimento do projeto final (TCC).
