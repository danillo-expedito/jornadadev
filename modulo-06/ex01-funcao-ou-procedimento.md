EXERCÍCIO Funçăo ou Procedimento?
====================================

Para cada item abaixo, a classificaçăo e justificativa em uma frase:

1. Str() — FUNÇĂO
   Justificativa: É uma funçăo porque processa um valor numérico e retorna uma nova string convertida para ser armazenada em variável ou utilizada em expressőes, sem realizar exibiçăo direta na tela.
2. QOut() — PROCEDIMENTO
   Justificativa: É um procedimento porque sua única finalidade é executar a açăo de exibir dados na tela com quebra de linha (equivalente ao comando ?), sem gerar ou devolver nenhum valor de retorno para o programa.
3. Date() — FUNÇĂO
   Justificativa: É uma funçăo porque consulta o relógio do sistema operacional e retorna um valor do tipo data (D) com a data atual para ser manipulado no código.
4. Len() — FUNÇĂO
   Justificativa: É uma funçăo porque analisa uma string ou um array e retorna um valor numérico inteiro indicando o tamanho ou a quantidade total de elementos que a estrutura possui.
5. Upper() — FUNÇĂO
   Justificativa: É uma funçăo porque recebe uma string como argumento e retorna uma nova string com todos os caracteres convertidos para letras maiúsculas, mantendo o valor original intacto.
6. ClearScreen() / Cls — PROCEDIMENTO
   Justificativa: É um procedimento pois apenas executa a açăo visual de limpar o console e posicionar o cursor no topo da tela, sem devolver qualquer valor de retorno ao script.
